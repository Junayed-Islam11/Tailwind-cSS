Setup

Get Tailwind running in minutes. Vite is the fastest path for new projects.

Install via npm (v3)
npm install -D tailwindcss postcss autoprefixer npx tailwindcss init -p
tailwind.config.js — content paths
// tailwind.config.js module.exports = { content: ["./src/**/*.{html,js,jsx,ts,tsx,vue}"], theme: { extend: {} }, plugins: [], }
globals.css — directives
@tailwind base; @tailwind components; @tailwind utilities;

Breakpoints

Tailwind is mobile-first. Prefix any class with a breakpoint to apply it at that width and above.

sm≥ 640px
md≥ 768px
lg≥ 1024px
xl≥ 1280px
2xl≥ 1536px
<!-- stacks on mobile, 3 columns on lg+ --> <div class="grid grid-cols-1 lg:grid-cols-3 gap-6"> </div> <!-- hidden on mobile, visible on md+ --> <nav class="hidden md:flex"></nav> <!-- smaller text on mobile, larger on xl --> <h1 class="text-2xl xl:text-5xl"></h1>

Dark Mode

Enable with darkMode: 'class' in config, then toggle the dark class on <html>.

config
darkMode: 'class', // or 'media' for system pref
toggle JS
document.documentElement .classList.toggle('dark')
Dark mode classes in practice
<body class="bg-white dark:bg-gray-900 text-gray-900 dark:text-gray-100"> <div class="bg-gray-100 dark:bg-gray-800 border border-gray-200 dark:border-gray-700 rounded-xl p-6"> <h2 class="text-gray-900 dark:text-white">Card</h2> </div>

Arbitrary Values

Escape the scale with square brackets. Works on any utility property.

Sizing & spacing
w-[342px] h-[calc(100vh-80px)] p-[13px] mt-[72px] top-[117px]
Colors & gradients
bg-[#1da1f2] text-[#ff6b6b] border-[#e2e8f0] from-[#667eea]
Grid
[grid-template-columns:2fr_1fr] col-span-[7]
Misc
text-[clamp(1rem,5vw,3rem)] rotate-[17deg] delay-[200ms]
CSS properties not in Tailwind yet
<!-- arbitrary CSS property --> <div class="[mask-image:linear-gradient(to_bottom,black,transparent)]"> </div> <!-- CSS variables --> <div class="bg-[var(--brand-color)]"> </div>

Pro Tips

@apply — reuse utilities in CSS
.btn-primary { @apply bg-blue-600 text-white px-5 py-2.5 rounded-lg font-semibold transition-colors hover:bg-blue-700; }
theme() in CSS
.hero { background: theme('colors.blue.600'); padding: theme('spacing.8'); font-size: theme('fontSize.xl'); }
Extend the theme
theme: { extend: { colors: { brand: '#6366f1', }, screens: { xs: '480px', }, fontFamily: { display: ['Sora', 'sans-serif'], }, } }
Safelist dynamic classes
// tailwind.config.js safelist: [ 'bg-red-500', 'bg-green-500', { pattern: /bg-(red|green|blue)-(400|500)/, } ]
Negative values
-mt-4 -mx-6 -top-2 -rotate-12 -translate-x-1/2
Important modifier
<!-- prefix with ! for !important --> <div class="!text-red-500 !font-bold"> </div>
